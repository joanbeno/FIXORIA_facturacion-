# FIXORIA — Invoice and Quote Templates

Static HTML templates for Fixoria-branded **quotes** and **invoices**. Built for print and PDF export.

## Files

| File | Description |
|---|---|
| `Fixoria_Cotizacion.html` | Quote template |
| `Fixoria_Factura.html` | Invoice template |

## Usage

Open in browser > File > Print > Save as PDF.
